## 6. Friend Request Flow

```mermaid
sequenceDiagram
    actor Alice
    actor Bob
    participant OOB as Out-of-band
    participant Server

    Alice->>OOB: Share pub_A
    OOB->>Bob: Bob receives pub_A

    Bob->>Server: blob = encrypt(pub_A, request)<br/>under hash(pub_A)
    Note over Server: Stores encrypted blob

    Alice->>Server: Connect (signed auth)
    Server->>Alice: PendingBlob
    Alice->>Alice: Decrypt, sees pub_B + request

    Alice->>Server: blob = encrypt(pub_B, accepted)<br/>under hash(pub_B)
    Alice->>Server: AckBlob (delete request)

    Bob->>Server: PendingBlob
    Bob->>Bob: Decrypt, friendship confirmed
```

## 7. P2P Connection Flow

```mermaid
sequenceDiagram
    participant Alice
    participant Server
    participant Bob

    Note over Alice,Bob: Both announce presence

    Alice->>Server: Announce(token_A, alice_ip)
    Bob->>Server: Announce(token_B, bob_ip)

    Server->>Bob: PeerOnline(token_A, alice_ip)

    Note over Alice,Bob: UDP hole punching (simultaneous)

    Bob->>Alice: UDP knock
    Alice->>Bob: UDP knock
    Note over Alice,Bob: NATs open ports

    Note over Alice,Bob: Authenticated session

    Alice->>Bob: ephemeral_pub_A + challenge_A
    Bob->>Alice: ephemeral_pub_B + Ed25519_sign(challenge_A)
    Alice->>Alice: Verify signature
    Alice->>Bob: Ed25519_sign(challenge_B)
    Bob->>Bob: Verify signature

    Note over Alice,Bob: session_key established<br/>chat starts
```
