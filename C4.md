# sechat - C4 Architecture Diagrams

## Level 1: System Context

```mermaid
C4Context
    title System Context - sechat

    Person(alice, "Alice", "Chat user with own device")
    Person(bob, "Bob", "Chat user with own device")

    System(sechat_alice, "sechat client (Alice)", "Desktop app with local encrypted storage")
    System(sechat_bob, "sechat client (Bob)", "Desktop app with local encrypted storage")

    System_Ext(server, "sechat server", "Blind presence + mailbox<br/>Sees only hashes and encrypted blobs")
    System_Ext(secret_share, "Secret Share App", "Out-of-band public key exchange")

    Rel(alice, sechat_alice, "Uses")
    Rel(bob, sechat_bob, "Uses")

    Rel(alice, secret_share, "Exchanges pub_key out-of-band")
    Rel(bob, secret_share, "Receives pub_key")

    Rel(sechat_alice, server, "WebSocket: presence, mailbox")
    Rel(sechat_bob, server, "WebSocket: presence, mailbox")

    Rel(sechat_alice, sechat_bob, "P2P encrypted (UDP)<br/>after hole punching")
```

## Level 4: Code/Sequence Diagrams

### Sequence: First-time Setup

```mermaid
sequenceDiagram
    actor User
    participant UI as UI (iced)
    participant Crypto as Crypto Module
    participant Disk as ~/.sechat/

    User->>UI: Open app first time
    UI->>UI: Check identity.key exists
    UI->>User: Show "Create new identity"
    User->>UI: Enter password
    UI->>Crypto: generate_x25519()
    UI->>Crypto: generate_ed25519()
    Crypto-->>UI: (x25519_priv, x25519_pub, ed25519_signing)
    UI->>Crypto: encrypt_keyfile(password, keys)
    Crypto->>Crypto: Argon2id(password, salt) → derived_key
    Crypto->>Crypto: ChaCha20-Poly1305 encrypt priv keys
    Crypto->>Disk: Write identity.key (124B)
    Crypto->>Disk: Write identity.pub (64B)
    Crypto-->>UI: Ok
    UI->>User: Show pub_key (to share out-of-band)
```

### Sequence: Login Flow

```mermaid
sequenceDiagram
    actor User
    participant UI as UI (iced)
    participant Crypto as Crypto Module
    participant Disk as ~/.sechat/

    User->>UI: Open app
    UI->>Disk: identity.key exists?
    Disk-->>UI: Yes
    UI->>User: Prompt password
    User->>UI: Enter password
    UI->>Crypto: read_keyfile()
    Crypto->>Disk: Read identity.key + identity.pub
    Disk-->>Crypto: FileData
    UI->>Crypto: decrypt_keyfile(password, FileData)
    Crypto->>Crypto: Argon2id(password, salt) → derived_key
    Crypto->>Crypto: ChaCha20-Poly1305 decrypt
    Crypto-->>UI: Keys { x25519_priv, ed25519_signing, ... }
    UI->>Crypto: load_peers()
    Crypto->>Disk: Iterate peers/
    Disk-->>Crypto: [PublicKey, ...]
    Crypto-->>UI: Peer list
    UI->>User: Show chat list
```

### Sequence: Add Friend (Out-of-Band + Server Mailbox)

```mermaid
sequenceDiagram
    actor Alice
    actor Bob
    participant AliceApp as Alice's sechat
    participant BobApp as Bob's sechat
    participant Server
    participant SecretShare as Secret Share App

    Note over Alice,Bob: Step 1: Out-of-band key exchange
    Alice->>SecretShare: Share pub_A as link
    SecretShare->>Bob: Bob opens link
    Bob->>BobApp: Add pub_A as friend

    Note over Alice,Bob: Step 2: Bob sends friend request
    BobApp->>BobApp: encrypt(pub_A, pub_B + "request")
    BobApp->>Server: SendBlob(hash(pub_A), encrypted_blob)
    Server->>Server: Store blob under hash(pub_A)

    Note over Alice,Bob: Step 3: Alice fetches and accepts
    AliceApp->>Server: Connect (signed auth)
    Server->>AliceApp: PendingBlob(blob_id, encrypted_blob)
    AliceApp->>AliceApp: decrypt(priv_A, blob) → pub_B + "request"
    AliceApp->>Server: AckBlob(blob_id)
    Server->>Server: Delete blob
    Alice->>AliceApp: Accept friend request
    AliceApp->>AliceApp: encrypt(pub_B, pub_A + "accepted")
    AliceApp->>Server: SendBlob(hash(pub_B), encrypted_blob)

    Note over Alice,Bob: Step 4: Bob fetches acceptance
    BobApp->>Server: (already connected)
    Server->>BobApp: PendingBlob(blob_id, encrypted_blob)
    BobApp->>BobApp: decrypt → "accepted"
    BobApp->>Server: AckBlob(blob_id)
    Note over Alice,Bob: Friendship established
```

### Sequence: P2P Connection & Chat

```mermaid
sequenceDiagram
    participant AliceApp as Alice's sechat
    participant Server
    participant BobApp as Bob's sechat

    Note over AliceApp,BobApp: Step 1: Both announce presence
    AliceApp->>AliceApp: token_A = HMAC(presence_key_AB, ts/15*15)
    AliceApp->>Server: Announce(token_A, alice_ip:port)
    BobApp->>BobApp: token_B = HMAC(presence_key_AB, ts/15*15)
    BobApp->>Server: Announce(token_B, bob_ip:port)

    Note over AliceApp,BobApp: Step 2: Discover peer is online
    Server->>BobApp: PeerOnline(token_A, alice_ip:port)

    Note over AliceApp,BobApp: Step 3: NAT traversal (UDP hole punch)
    BobApp->>AliceApp: UDP "knock" to alice_ip:port
    AliceApp->>BobApp: UDP "knock" to bob_ip:port
    Note over AliceApp,BobApp: Both NATs open ports

    Note over AliceApp,BobApp: Step 4: Session establishment
    AliceApp->>AliceApp: Generate ephemeral X25519 keypair
    BobApp->>BobApp: Generate ephemeral X25519 keypair
    AliceApp->>BobApp: ephemeral_pub_A + challenge
    BobApp->>AliceApp: ephemeral_pub_B + Ed25519_sign(challenge)
    AliceApp->>AliceApp: Verify signature with bob_verifying_key
    AliceApp->>BobApp: Ed25519_sign(bob's challenge)
    BobApp->>BobApp: Verify with alice_verifying_key
    Note over AliceApp,BobApp: session_key = HKDF(ECDH(ephemerals), "session")

    Note over AliceApp,BobApp: Step 5: Chat
    AliceApp->>BobApp: ChaCha20-Poly1305(session_key, "Hello Bob")
    BobApp->>BobApp: Decrypt, store with storage_key
    BobApp->>AliceApp: ChaCha20-Poly1305(session_key, "Hi Alice")
```

### Sequence: Message Storage

```mermaid
sequenceDiagram
    participant Client as sechat client
    participant Crypto as Crypto Module
    participant Sled as sled DB
    participant Disk as peers/<hash>/chat.db

    Note over Client: New message received
    Client->>Crypto: load_storage_key(peer_pub, identity_priv)
    Crypto->>Crypto: HKDF(ECDH, "storage") → storage_key
    Crypto-->>Client: storage_key

    Client->>Client: Build Message {author, ts, text}
    Client->>Client: MessagePack serialize
    Client->>Crypto: encrypt(storage_key, nonce(counter), plaintext)
    Crypto-->>Client: ciphertext

    Client->>Sled: insert(counter.to_be_bytes(), ciphertext)
    Sled->>Disk: Persist

    Note over Client: Read history
    Client->>Sled: iter()
    loop For each (key, value)
        Sled-->>Client: (counter_bytes, ciphertext)
        Client->>Crypto: decrypt(storage_key, nonce(counter), ciphertext)
        Crypto-->>Client: plaintext
        Client->>Client: MessagePack deserialize → Message
    end
```
